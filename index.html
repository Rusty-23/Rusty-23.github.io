Subject Information

This course focus on the basic web-based design and application of web developing. This course is divided into two parts. First it covers the different elements in HTML and second its cover the basic parts of JavaScript. This course includes classroom lectures and hands-on computer laboratory and exercises.

Subject Timeline

Week 1 - 3

Demonstrate familiarity with the different area and tools of HTML

Identifying the importance, use and different tags of HTML

Week 4 - 5

Demonstrate familiarity in different HTML formatting.

Identify the use of tags for Unordered List, ordered list and list item

Week 6 - 7

Demonstrate competence in creating programs using Paragraph formatting and marquee structure of HTML script.

Creating a program with web paragraph formatting

Week 8 - 10

Demonstrate competence in creating programs using the Hyperlink, Inserting images, forms and tables in HTML script

Week 11-15

Demonstrate familiarity in using JavaScript basic code, Control Structures and Looping Statement
